## 项目介绍

**PS图层批量替换导出** 是一款面向设计师、自媒体创作者和批量处理需求的桌面工具。它能够基于 Photoshop PSD 模板，自动识别可替换的图层，将指定文件夹中的照片批量替换进去，并导出为 PNG 格式的封面图片。

核心解决了一个长期痛点：**当需要为几十甚至上百张照片套用同一个 PSD 模板时，手动在 Photoshop 中逐张替换、调整、导出极其耗时**。本工具将这一过程自动化，从数小时缩短到几分钟。

## 解决了什么痛点

| 痛点 | 传统方式 | 本工具 |
|------|---------|--------|
| 批量套用模板 | 逐张在 PS 中打开、替换、导出 | 一键批量处理 |
| 智能图层识别 | 手动查找替换图层 | 自动检测图片图层和剪贴蒙版 |
| 实时预览调整 | 无法预览，导出后才能看效果 | 拖拽实时预览，所见即所得 |
| 中文文件名支持 | 部分工具不支持中文 | 完整支持中文路径和文件名 |
| 自然排序 | 1, 10, 11, 2 的字典序混乱 | 1, 2, 3... 10, 11 智能排序 |
| 输出管理 | 手动选择输出目录 | 自动在照片目录下创建"封面"文件夹 |

## 功能特性

- **智能图层检测** — 自动识别 PSD 中的图片图层，支持剪贴蒙版（Clipping Mask）关系
- **实时预览** — 拖拽调整位置和缩放，60fps 流畅预览
- **批量导出** — 一次导出所有照片套用模板后的 PNG 封面
- **自然排序** — 文件名按数字顺序排列（1→2→3→...→10），而非字典序
- **配置记忆** — 自动记住上次使用的 PSD、照片目录和输出目录
- **键盘导航** — 左右箭头快速切换缩略图
- **滚轮缩放** — 点击预览区后用滚轮缩放素材
- **高分辨率输出** — 导出时自动按原始 PSD 分辨率渲染

## 使用教程

### 1. 准备 PSD 模板
在 Photoshop 中设计好封面模板，确保有一个图层用于放置照片（支持普通图层和剪贴蒙版）。

### 2. 运行软件
双击便携版或安装后从开始菜单/桌面快捷方式启动。

### 3. 导入模板和照片
- **PSD 模板**：点击"浏览"选择 PSD 文件
- **照片文件夹**：点击"浏览"选择包含照片的文件夹（支持 jpg/png/webp 等格式）

### 4. 调整效果
- 点击底部缩略图预览每张照片的效果
- 在右侧预览区拖拽调整照片位置
- 点击预览区后，用滚轮缩放照片
- 按住素材边缘拖动调整大小

### 5. 批量导出
点击标题栏右侧的"批量导出 PNG"按钮，软件会自动在照片文件夹内创建"封面"子目录并导出所有封面。
